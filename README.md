# Real-Time Hand Gesture Recognition using SVM and Computer Vision

## Overview

This project demonstrates a real-time hand gesture recognition system using Computer Vision, MediaPipe, and Support Vector Machine (SVM). The system detects hand landmarks from a webcam feed and classifies gestures such as:

* Fist
* Open Palm
* Thumbs Up
* Victory Sign

The project was developed as part of the Principles of Artificial Intelligence coursework at NMIMS.

---

## Technologies Used

* Python
* OpenCV
* MediaPipe
* scikit-learn
* NumPy

---

## Methodology

1. Capture webcam frames
2. Detect hand landmarks using MediaPipe
3. Extract landmark coordinates as features
4. Train SVM classifier
5. Predict gesture in real time

---

## Features

* Real-time gesture recognition
* Landmark-based feature extraction
* Lightweight SVM classifier
* Webcam-based interaction
